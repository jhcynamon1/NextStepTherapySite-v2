## üí° KEY INSIGHTS FROM THIS PROCESS

### What We Learned:

1. **‚úÖ Ahrefs Can Be Wrong:**
   - Ahrefs classified keywords as "informational"
   - But they're actually "hybrid informational-transactional"
   - Trust GSC query data over Ahrefs intent classification

2. **‚úÖ Competitor Analysis Matters:**
   - Inner Oak has separate pages per service (anxiety, depression, PTSD)
   - They DON'T have multiple pages for same service
   - We copied their structure: anxiety-specific + general hub

3. **‚úÖ Parent Topics Are Critical:**
   - "depression therapy burlington" parent topic ‚Üí "anxiety therapy burlington"
   - This tells us they're related and should link to each other
   - Future: Create depression page and link to anxiety page

4. **‚úÖ GSC Query Data Reveals Truth:**
   - anxiety-therapy-burlington was getting wrong traffic
   - Simple content refocus should fix this
   - File name ‚â† content focus = Google gets confused

5. **‚úÖ Word Count Sweet Spot:**
   - Not 1,200 (too short for therapy trust-building)
   - Not 8,000 (too long, dilutes focus)  
   - 2,500-4,000 seems optimal for therapy pages
