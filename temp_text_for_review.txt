                    It's normal to have questions before starting therapy for depression. Here are answers to some common concerns.
                    
                    "I'm still functioning at work. Is my depression 'bad enough' for therapy?"
                    
                    
                        High-functioning depression is real.
                        Many successful professionals manage to maintain their performance while struggling internally. You don't have to wait for a crisis to seek support. Therapy can help you move from just 'functioning' to truly 'thriving'.
                    
                    
                    "Will we just talk about my past?"
                    
                    
                        Our focus is on the present and future.
                        While your history is important, our primary focus will be on the practical skills you can learn and actions you can take right now to improve your life, aligned with the future you want to create.
                    
                    
                    "How is ACT different from other therapies for depression?"
                    
                    
                        It's about action, not just analysis.
                        Instead of trying to eliminate negative thoughts, ACT teaches you to change your relationship with them. It helps you take meaningful, values-driven action even when feelings of depression are present, which is a powerful way to break the cycle.

        // Smart sticky header functionality
        document.addEventListener('DOMContentLoaded', function() {
            const header = document.querySelector('.header.smart-sticky');
            if (!header) return;
            
            let lastScrollY = window.scrollY;
            let isHeaderVisible = true;
            const headerHeight = header.offsetHeight;
            const scrollThreshold = 10; // Minimum scroll distance to trigger
            
            function updateHeader() {
                const currentScrollY = window.scrollY;
                const scrollDifference = Math.abs(currentScrollY - lastScrollY);
                
                // Only proceed if scroll difference is significant
                if (scrollDifference < scrollThreshold) {
                    return;
                }
                
                if (currentScrollY <= headerHeight) {
                    // At the top of the page
                    header.style.transform = 'translateY(0)';
                    isHeaderVisible = true;
                } else if (currentScrollY > lastScrollY && isHeaderVisible) {
                    // Scrolling down - hide header
                    header.style.transform = `translateY(-${headerHeight}px)`;
                    isHeaderVisible = false;
                } else if (currentScrollY < lastScrollY && !isHeaderVisible) {
                    // Scrolling up - show header
                    header.style.transform = 'translateY(0)';
                    isHeaderVisible = true;
                }
                
                lastScrollY = currentScrollY;
            }
            
            // Use requestAnimationFrame for smooth performance
            let ticking = false;
            function requestTick() {
                if (!ticking) {
                    requestAnimationFrame(updateHeader);
                    ticking = true;
                }
            }
            
            window.addEventListener('scroll', function() {
                requestTick();
                ticking = false;
            }, { passive: true });
            
            // Set initial styles
            header.style.transition = 'transform 0.3s cubic-bezier(0.4, 0, 0.2, 1)';
            header.style.transform = 'translateY(0)';
        });
