## Emergency Contact (If Things Break)

**If indexing stops completely:**
1. Check vercel.json for redirect issues
2. Verify robots.txt isn't blocking Google
3. Resubmit sitemap to GSC
4. Use URL Inspection tool to force crawl

**If schema breaks:**
1. Validate at: https://search.google.com/test/rich-results
2. Check for JSON syntax errors
3. Ensure schema is in <script type="application/ld+json">

**If traffic drops suddenly:**
- Check GSC for manual actions (Security & Manual Actions)
- Check GSC for coverage errors
- Verify site is still live (visit nextsteptherapy.ca)
