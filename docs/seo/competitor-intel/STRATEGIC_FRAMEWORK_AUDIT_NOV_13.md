### 2. Framework's Testimonial Strategy Needs CRPO Adaptation

**Issue:** Framework includes testimonials but examples lack proper attribution (CRPO requirement).

**Solution:**
- ALL testimonials must include first name + initial + general context
- Add disclaimer: "Individual experiences, results may vary"
- Obtain written client permission
- Focus on "process" language (developed skills) not "outcome" language (cured anxiety)

**Format:**
```
"Jesse helped me develop practical strategies for managing panic attacks during client presentations. The ACT techniques work even when I'm in the moment."
‚Äî Sarah M., Tech Project Manager, Toronto
```

**Action:** Create testimonial collection process + approval workflow before implementing.

---

### 3. Don't Standardize What's Working

**Critical Finding:** Homepage and Workplace Stress pages are performing well despite not matching framework exactly.

**Why This Matters:**
- Framework is based on general therapy marketing, not Jesse's specific positioning
- Jesse's "no therapy BS" / straightforward approach has found its voice
- Forcing framework structure could hurt conversion on pages that work

**Recommendation:**
- Mark homepage and workplace-stress page as "template pages" (what works now)
- Use framework for NEW pages and UNDERPERFORMING pages
- A/B test framework changes on working pages before full implementation
- Measure, don't assume

---

### 4. Content Length Debate: Framework vs. SEO Reality

**Framework:** 2,000-2,500 words per page  
**Reality:** Virtual-therapy-ontario.html is 8,000+ words and likely ranking well

**Analysis:**
- Longer content often ranks better for competitive keywords
- But longer content can hurt conversion if not scannable
- Framework prioritizes conversion, existing page prioritizes ranking

**Solution:**
- Keep long-form content for competitive keywords (virtual therapy ontario, anxiety therapy ontario)
- Add table of contents / jump links for UX
- Use framework length (2,000-2,500) for city pages where local signals > content length

### 5. The "Casual vs. Professional" Tone Decision

**Framework Provides Two Templates:**
1. **Standard:** Professional, clear, authoritative
2. **Casual:** "You're crushing it but..." conversational

**Current Pages:** Mostly professional with conversational elements (hybrid)

**Recommendation:**
- Use **Standard Template** for: Depression, trauma, crisis-related pages
- Use **Casual Template** for: Toronto tech/startup, Waterloo, workplace stress pages
- Use **Hybrid (Current Approach)** for: Most pages - it's working well
- Test casual hero variants on Toronto/Ottawa professional pages
