## Archiving Rules

Move documents to archive when:
- Task is completed
- Document is superseded by a newer version
- More than 30 days old without updates
- No longer relevant to current work

Keep documents in root/current when:
- Actively being used
- Reference material (guides, checklists)
- Single source of truth for a topic
